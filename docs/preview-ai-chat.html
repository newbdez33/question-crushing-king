A company hosts a static website on Amazon S3 and wants to serve it globally with low latency, HTTPS, and the ability to invalidate cached objects on demand. Which combination of AWS services should the solutions architect recommend?

A
S3 + AWS Global Accelerator
B
S3 + Amazon CloudFront + ACM certificate
C
S3 + Route 53 latency-based routing only
D
EC2 behind an Application Load Balancer

Correct answer ✓

Correct: B

Your answer: B

AI Explanation gpt-4o-mini
Please explain why B is correct and why the others are wrong.
Why B is correct. Amazon CloudFront is a CDN that fronts your S3 origin, terminates TLS using an ACM certificate, and supports on-demand cache invalidation. That covers the three requirements: global low latency, HTTPS, and invalidate cached objects.

Why the others are wrong.
  • A. Global Accelerator routes TCP/UDP to regional endpoints; it does not cache static content or expose invalidation.
  • C. Route 53 latency-based routing only chooses an endpoint — it doesn't cache or provide HTTPS for S3 websites.
  • D. EC2 + ALB is over-provisioned for static content and lacks edge caching.
